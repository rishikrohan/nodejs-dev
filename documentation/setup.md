# Nodejs Installation (MacBook)

## Download the install script

`curl -sL https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.0/install.sh -o install_nvm.sh`

## Run the install script

`bash install_nvm.sh`

`zsh install_nvm.sh`

## Restart the terminal

`source ~/.zshrc`

## Install Nodejs

`nvm install --lts`

## Verify if it worked

`node -v`
